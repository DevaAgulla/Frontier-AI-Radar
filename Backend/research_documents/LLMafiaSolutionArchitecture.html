🧠 3-Layer Memory Architecture — Wired Into Every Agent's ReAct Loop
⚡ Short-Term Memory
Stores all findings from the current run only.

Agents read before acting to avoid duplicate fetches within the same run.

Automatically cleared after the run completes.
Implementation: LangGraph State (in-process TypedDict)
📚 Long-Term Memory
Stores historical findings, past digests, and trend baselines across runs.

Strategy Planner reads to adjust today's priorities.

Digest Agent reads to write "What changed since yesterday."
Implementation: PostgreSQL + pgvector index
🏷️ Entity Memory
Tracks known organizations, models, benchmarks, and papers with embeddings.

Every agent reads entity context before summarizing a finding.

Enables awareness like: "GPT-5 is OpenAI's 5th generation model."
Implementation: ChromaDB (vector embeddings) + Postgres
